# Python Adventure Game

This project is a text-based adventure game developed using Python.
The player explores different paths and makes decisions to find a legendary treasure.

## Project Objective
To practice core Python concepts such as:
- Variables and user input
- Conditional statements
- Loops
- Functions

## How the Game Works
- The player starts the game by entering their name
- They choose between a forest path or a cave path
- Each choice leads to a different outcome
- The game can end in a win, loss, or restart

## How to Run the Game
1. Ensure Python 3 is installed
2. Open a terminal in the project folder
3. Run the following command:

```bash
python adventure_game.py
